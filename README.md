# Crypton

```
Демо приложения - агрегатора криптовалютных кошельков.

Стек: Vue.js, Vuex, Vue-CLI.

Мобильной версии нет.
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
