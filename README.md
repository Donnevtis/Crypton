# Cryptonix

**Cryptocurrency aggregator web application demo.**

### Stack

- Vue.js
- Vuex
- Vue-Router

### Link

**[cryptonix.surge.sh](http://cryptonix.surge.sh/)**

## Project setup

`yarn install`

### Compiles and hot-reloads for development

`yarn run serve`

### Compiles and minifies for production

`yarn run build`

### Lints and fixes files

`yarn run lint`
